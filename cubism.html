An attempt to create a "cubism-like" interactive 3d rendered space.
(My final project in computer games cource). there are 5 diffrent items in space: chair, table, bottle, glass and a teapot. each item is visualy represented throw 9 tiles. each tile displays the item from different angles according to the position of the camera. each angle is rendered with the framebuffer into texture. after rendering each item to nine texture of different point of views , i placed them in space facing always to the camera, and randomly changing there shape. and after all that i added another post processing to all of the scene, adding canvas and manipulating colors. for each item i added a silhouette, for painting like feeling.

Build with: c++, openGL, openAL, cg. computer games course, huji 2013.
